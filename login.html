Got Questions?

Everything you need to know about Campus Finder.

Is it free to use? +
Yes! Campus Finder is a community platform built by students, for students. It is 100% free.
How is ownership verified? +
We encourage finders to ask for specific details not mentioned in the report (like a wallpaper description or a specific item inside a bag).
Where should I meet the owner? +
Always meet in public campus locations like the main canteen, library entrance, or security desks.
